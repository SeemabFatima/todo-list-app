# 📋 To-Do List App

This is a console-based To-Do List application I created during my **DecodeLabs Python Internship**. The goal of this project was to practice Python fundamentals by building a program that can manage daily tasks and save them in a JSON file.

## Author

**Seemab Fatima**  
Python Programming Intern – DecodeLabs

## Features

- Add new tasks
- View all saved tasks
- Search tasks by keyword
- Edit task titles
- Mark tasks as Pending or Completed
- Delete tasks by ID
- Automatically save tasks using a JSON file
- Store the date and time each task is created

## Technologies Used

- Python 3
- JSON
- `datetime`
- `os`

## ▶️ Running the Project

1. Install Python 3 if it's not already installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python todo.py
```
